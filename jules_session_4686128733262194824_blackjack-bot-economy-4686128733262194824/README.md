# Blackjack Bot

Это Telegram-бот для игры в блэкджек и рулетку с локальной экономикой для каждого чата.

## Инструкция по запуску

### Шаг 1: Настройка базы данных Firebase (Firestore)
1. Перейдите в [Firebase Console](https://console.firebase.google.com/).
2. Нажмите **Add project** (Добавить проект), введите название (например, BlackjackBot) и нажмите Continue. Аналитику (Google Analytics) можно отключить.
3. После создания проекта перейдите в левом меню в раздел **Build -> Firestore Database**.
4. Нажмите **Create database** (Создать базу данных).
   - Выберите регион (например, `eur3 (Europe)`).
   - Выберите режим **Start in production mode** или **Start in test mode** (неважно для бота, так как мы будем использовать сервисный аккаунт).
   - Нажмите **Enable**.
5. Теперь нужно получить ключ для подключения бота:
   - В левом верхнем меню нажмите на шестеренку рядом с "Project Overview" и выберите **Project settings** (Настройки проекта).
   - Перейдите во вкладку **Service accounts** (Сервисные аккаунты).
   - Нажмите кнопку **Generate new private key** (Сгенерировать новый закрытый ключ) и подтвердите.
   - На ваш компьютер скачается файл формата `.json`.

### Шаг 2: Настройка бота
1. Переименуйте скачанный `.json` файл в `firebase-key.json` и поместите его в ту же папку, где находится этот файл `README.md`.
2. Создайте файл `.env` в этой же папке и добавьте в него следующие строки:
   ```env
   BOT_TOKEN=YOUR_BOT_TOKEN_HERE
   FIREBASE_KEY_PATH=firebase-key.json
   CREATOR_USERNAME=z_1l1
   ```
   *Убедитесь, что в `BOT_TOKEN` указан ваш актуальный токен бота из BotFather, а в `CREATOR_USERNAME` — ваш username в Telegram (без @).*

### Шаг 3: Запуск на локальном компьютере или сервере
1. Установите Python (версии 3.9 и выше).
2. Откройте терминал (или командную строку) в папке с ботом.
3. Установите необходимые библиотеки командой:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите бота командой:
   ```bash
   python src/main.py
   ```
