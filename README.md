# Бот Экономики и Казино для Telegram

Бот для групп и личных сообщений с мини-играми, магазином и системой VIP.

## Как запустить:
1. Скачайте файлы проекта.
2. Откройте терминал (консоль) в папке с проектом.
3. Установите зависимости: `pip install -r requirements.txt`
4. Скопируйте файл `.env.example`, переименуйте его в `.env` и впишите туда свои данные (токен бота и ваш юзернейм).
5. Положите скачанный из Firebase ключ в корень папки под названием `firebase-key.json`.
6. Запустите бота командой: `python src/main.py`
